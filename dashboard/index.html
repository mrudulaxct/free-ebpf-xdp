What To Explain

FRER goal: send duplicate frames through independent paths so one path can fail without service interruption.

Replication: XDP reads VLAN 100, inserts a 6-byte R-tag sequence number, then broadcasts the frame to ab0 and ab1.

Elimination: the first sequence number is redirected to the destination; later copies with the same sequence are counted as duplicates and dropped.

Kernel/userspace split: userspace loads programs and fills BPF maps; packet decisions happen in the kernel at XDP speed.
